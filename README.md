# Backend 

빌리지의 종합적인 데이터와 IoT 기기로 부터의 데이터를 처리하는 백엔드 서버



## Env

| 환경         | 버전   |
| ------------ | ------ |
| Node         | 15.5.0 |
| Express      | 4.17.1 |
| Mongoose     | 5.12.1 |
| jsonwebtoken | 8.5.1  |



## Install 

```shell
git clone https://github.com/TeamMartini/hms_be.git
cd hms_be
yarn install # or npm install
PRODUCTION_MONGO_DB={YOUR_MONGO_DB_URL} node app
```
